# NextJs Blog

This is a simple, static, NextJs blog featuring Remote MDX. The porpuse of this project is to paste text easily on my classes so the students don't have to download any files and can read it using their smartphones.

To add new blog posts, simply create a .mdx file on the `src/content` directory considering at least title, description and publish date.

## Features

- Full Markdown support for documents with Remote MDX.
- TailwindCSS Styling.
- ShadcnUI Components.
- NextJs 14 with App Router.
- Deployment to Github using Github Actions.
